# dev-superior-crud-client-spring
## Complete CRUD project (Create, Retrive, Update, Delete) whith REST web services to access a Client resource:

- Paged feature search;
- Resource search by Id;
- Insert new resource;
- Update resource;
- Delete resource;

### Technologies:

- Java11
- Spring Boot 2.4.5
- H2 Database
- Spring Data JPA
